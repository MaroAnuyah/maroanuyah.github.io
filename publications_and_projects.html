Projects

Research Projects

Popularity and Demographic Biases in Recommender Evaluation and Effectiveness

We conducted a systematic investigation on the quality and effectiveness of Recommender Systems, using offline evaluation and a utility-based metric of recommendation effectiveness to explore whether different user demographic groups experience similar recommendation accuracy. We find demographic differences in measured recommender effectiveness across two data sets containing different types of feedback in different domains.


Looking for the movie Seven or Sven from the movie Frozen? A multi-perspective strategy for recommending queries for children

We designed an algorithm with the aim of providing query suggestions that best captures the queries that children can use to locate resources relevant to them. It is based on a multi-perspective recommendation strategy that uses a wide and deep neural network for identifying patterns in the queries to see if they are child-like, and also uses a semantic similarity strategy for recommending diverse queries (queries that capture different topics).


Can Readability Enhance Recommendations on Community Question Answering Sites?

Initial examination on the impact of text complexity in the recommendation process of community question answering sites. The results show the benefits of incorporating readability information in this domain.

Class Projects

Summarizing Amazon Product Reviews

Class project that involves using a text summarization technique for generating overall review summary using existing reviews written by Amazon product buyers. For this task, different feature extraction, opinion-mining, and sentiment analysis techniques were explored. A Naive Bayes classifier was used for training and testing features

Kaggle Competition: Predicting Similarity between two questions using the Quora Question Pairs dataset

Class project that involves using different information retrieval and natural language processing techniques for predicting similarity between two given questions. For this purpose, different similarity measures, such as binary, weighted and semantic similarity, and Word2Vec embeddings of each question, are used as input to a random forest classifier.
